# Shanion

`Shanion` 是一个多端本地优先碎片记录客户端。

## 当前目标

- 以本地存储为核心，记录不会依赖远端服务。
- 支持文本、链接、图片、视频、语音、文件等碎片内容。
- 每条记录围绕文本内容与附件列表组织，并自动记录创建时间、位置与设备信息。
- 提供检索、同步、智能整理等能力。
